generator page
* сторінка має відкриватися при переході на /generator
* для генерації кольорів використати відповідну функцію з папки shared
* при натисканні на "пробіл" кольори мають мінятися
* при натисканні на save\"серце" необхідно зафарбувати його колір у чорний. При повторному натисканні нічого не відбувається
* назви кольорів можна захаркодити
* при наведенні на область між кольорами, має з'являтися знак "+"
** при натисканні на "+" кількість кольорів має збільшитися, але не більше 7
* при наведені на колір мають з'являтися іконки, ніякого функціоналу не несуть


special color page
* має відкриватися на шляху /colors/:colorHex
* що б знати який колір відображати, використати значення параметра colorHex
* у секції conversion поля HEX, RGB, CMYK, HSB, HSL мають залежати від кольору який був відкритий. Решта полів будуть захардкоджені


main page
* при натисканні на кнопку "Start Generator", користувача має перекинути на /generator
* Елемент color of the day має мати змінювати своє значення кожного разу коли сторінка оновлюється
* при натисканні на color of the day користувача має перекинути на /colors/:colorHex де colorHex поточне значення кольору який відображається
